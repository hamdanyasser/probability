## üéØ STUDY STRATEGIES BY TOPIC

### Mastering Combinatorics
1. **Always** start by asking: "Does order matter?"
2. Draw small examples (n=2,3) to verify your formula
3. Use the urn models table as quick reference
4. Practice, practice, practice ‚Äî these are mechanical

### Mastering Probability
1. For uniform models: Convert to counting
2. Draw pictures: trees for sequential, Venn for unions
3. Always check: Is probability between 0 and 1?
4. Memorize: Complement for "at least one"

### Mastering Conditional Probability
1. Write definition: $P(A|B) = \frac{P(A \cap B)}{P(B)}$
2. LOTP when you can't get $P(A)$ directly
3. Bayes when you need to flip the conditional
4. Independence: verify before multiplying

## üö® COMMON PITFALLS (Read Before Exam!)

‚ùå **Don't assume independence** ‚Äî verify it!
‚ùå **Don't confuse disjoint with independent** ‚Äî they're opposite!
‚ùå **Don't forget the overlap** in $P(A \cup B)$
‚ùå **Don't mix up** $P(A|B)$ and $P(B|A)$
‚ùå **Don't use** $\binom{n+r}{r}$ for stars and bars ‚Äî it's $\binom{n+r-1}{r}$!
‚ùå **Don't forget** "without replacement" changes probabilities
‚ùå **Don't skip** showing your work ‚Äî partial credit matters!
